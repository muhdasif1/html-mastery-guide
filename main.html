11. Common HTML Attributes

Attribute Description Example
id Unique identifier <div id="header">
class Class name for CSS <p class="text">
style Inline CSS <p style="color:red">
title Tooltip text <p title="Info">
href Link URL <a href="url">
src Source file <img src="image.jpg">
alt Alternative text <img alt="desc">
width/height Dimensions <img width="100">
target Link target <a target="_blank">
disabled Disable element <input disabled>
required Required field <input required>
placeholder Input hint <input placeholder="text">
readonly Read only <input readonly>
maxlength Max characters <input maxlength="10">

12. Block vs Inline Elements

Block Elements (Take full width)

div, p, h1-h6, ul, ol, li, table, form, header, footer, section, article, nav, aside

Inline Elements (Only take necessary width)

span, a, img, strong, em, b, i, u, br, input, button, label, select, textarea
